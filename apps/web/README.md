# Web App

Formal frontend home for the LIFF/PWA experience.

Current Alpha HTML prototypes remain in the repository root and in `deploy/cloudflare-alpha`.
This app will progressively replace those prototypes with a normal frontend build.

Responsibilities:

- LIFF SDK initialization.
- Quiz UI and local interaction state.
- API calls to `apps/api`.
- Result, share card, and AI crest wall presentation.
- No service-role secrets and no direct privileged database writes.
