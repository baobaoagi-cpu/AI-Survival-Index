# Alpha-13 Quiz Choice Clarity UX

## 目的

讓測驗頁在 390px 手機畫面上一眼看懂：

1. 現在是第幾題
2. 上方是情境題目
3. 下方是三選一選項
4. 每張卡片都可以點擊

這一版只改測驗頁 UX，不改計分、不改 API、不改資料庫。

## UX 規則

測驗頁必須分成三個明確層級：

- 題號：`QUESTION 1 / 6` 與 `三選一`
- 題目：獨立情境卡，標示 `情境題`
- 選項：明確提示 `你第一個反應是？`，並顯示 A / B / C

每個選項卡必須包含：

- A / B / C 視覺標記
- 情緒標籤
- 選項句
- 明確 CTA：`選這個 →`
