# Option Image Prompt Bible

This document defines the 18 option images for the Alpha quiz.

Use these prompts with ChatGPT image generation or another image model. Keep every image consistent with the same visual system.

## Global Style

Apply this style to every option image:

```text
16:9 cinematic option card illustration, 768x432 px, techno-mystical atmosphere, warm futuristic lighting, abstract symbolic scene, no readable text, no logo, no UI, no human face close-up, not cartoon, not anime, not comic, high contrast but not too dark, clear central silhouette, elegant spiritual technology, subtle circuit patterns, gold and teal energy accents, deep violet black background, premium tarot-like visual language, suitable for a 390px mobile quiz card, central subject inside safe 70 percent area, clean composition, emotional and atmospheric
```

Negative prompt:

```text
no words, no letters, no numbers, no watermark, no brand logo, no speech bubble, no realistic celebrity, no clutter, no low contrast, no horror, no gore, no childish cartoon, no anime, no comic book panel, no busy background, no tiny details that disappear on mobile
```

## Output Specs

```text
Aspect ratio: 16:9
Target size: 768x432 px
Minimum acceptable size: 640x360 px
Format: PNG or WebP
Color space: sRGB
Safe area: central 70 percent
File path pattern: assets/scenes/options/scene-XX-option.png
```

## Consistency Checklist

Before accepting each image:

- It is 16:9.
- It has no readable text.
- The subject is clear at small mobile size.
- It feels like the same visual family as the other 17 images.
- It is not a literal cartoon scene.
- It communicates the choice emotionally within one second.
- It leaves enough dark space and edge padding for mobile cropping.
