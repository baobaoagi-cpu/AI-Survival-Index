# Option Image Prompt Bible

This document defines the 18 quiz option illustrations for the Alpha quiz.

All option images must follow:

1. `docs/option-bible-spec-v1.md`
2. `docs/archetype-bible-v1.md`

Images are not personality definitions. They are visual translations of the option's dimension effect, user emotion, and mass-market psychology cue.

## Alpha-18 Direction

The option images are now grounded, humorous, and easy to understand for general users.

Core direction:

- Everyday work and life metaphors
- Funny but not childish
- Mass-market economic psychology
- Large readable iconography
- AI/futuristic accent lines
- No tiny details that disappear inside the mobile option thumbnail
- No long text inside the image

The semantic names live in `assets/scenes/options/manifest.json`. The PNG filenames stay stable so the UI does not break.

## Output Specs

```text
Aspect ratio: 1:1
Target size: 512x512 px
Format: PNG
Color space: sRGB
Safe area: central 70 percent
Display target: small cropped thumbnail inside a 390px mobile quiz option card
File path pattern: assets/scenes/options/scene-XX-a.png
```

## Global Visual Prompt

Use this base prompt when regenerating with ChatGPT image generation:

```text
Square mobile quiz option illustration, 512x512 px, grounded everyday metaphor, funny but not childish, suitable for Taiwanese LINE users, mass-market economic psychology, one clear central icon, warm futuristic lighting, dark violet mobile-game background, subtle AI circuit accents, gold highlight lines, high contrast, clean composition, readable at small thumbnail size, no long text, no UI buttons, no watermark, no brand logo, no anime, no comic panel, no human face close-up.
```

Negative prompt:

```text
no clutter, no tiny unreadable details, no paragraph text, no watermark, no brand logo, no horror, no gore, no childish cartoon, no anime, no celebrity, no busy background, no low contrast
```

## Regeneration Rules

When regenerating:

1. Keep the existing filenames.
2. Keep the main object large and centered.
3. Avoid long text inside the image.
4. Use the semantic name and psychology cue as prompt intent, not as visible label.
5. Test inside the real 390px quiz card before accepting the image.

## Local Generator

The current deterministic Alpha-18 placeholders are generated by:

```text
scripts/generate-option-card-assets.py
```

This generator is not the final image factory. It exists to keep the UI coherent before the final ChatGPT Image / MCP image pipeline is connected.
